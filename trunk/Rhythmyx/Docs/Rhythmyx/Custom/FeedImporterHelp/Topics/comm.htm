Configuring Rhythmyx for feed importer The feed importer must now be associated with the desired Communities.
  1. Go to the Content Manager and select Package Manager.

  1. Select PSO Feed Importer.
  2. Click on the tab labeled Visibility.
  3. The following screen appears:
  1. Select PSO Feed Importer and click Edit Associations.
  2. Select the desired Available Communities and click OK.

  1. Return to the Content Manager
