Installing the Feed ImporterThe first step to installing the Feed Importer is to install the PSO Feed Importer Package; the Rhythmyx Server must be running. The following steps are required:
  1. Save PSOFeedImporter.ppkg to Desktop
  2. Run the Percussion Package Installer

  1. Select the path for the PSO Feed Importer from your Desktop and enter your Host, Port, and User ID information.
  2. Select Install.
  3. Should a Validation Error or Warning occur, select Continue.

  1. The above screen will appear once the installation is complete; select Exit.


Once the PSO Feed Importer package has been installed, stop the Rhythmyx Server in order to install the Content-rs and PSOToolkit6.7 folders.
  1. Copy both folders to c://Rhythmyx/Patch


  1. For each folder, run the Install.bat file.
  2. Upon build completion, restart the Rhythmyx Server.
